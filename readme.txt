== Description ==

If a page has a custom field named staticpagetemplate, only the page content(html) is sent to browser.

== Installation ==

Installation is easy:

* Download the plugin.
* Copy the `static-page-template` directory to the plugins directory of your blog.
* Enable the plugin in your admin panel.
